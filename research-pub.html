Path Planning of Mobile Robot using Adaptive Particle Swarm Optimization

In this research project, we developed a novel path planning algorithm for mobile robot navigation using Adaptive Particle Swarm Optimization. The algorithm was able to find the shortest path at the same time avoiding the collision with obstacles. We used e-puck robot from Webots simulator as our testbed for this research project.

Himanshu, A. Singha, A. Kumar, and A. K. Ray, "Path Planning of Mobile Robot using Adaptive Particle Swarm Optimization", in 2nd International Conference on Innovations in Computational Intelligence and Computer Vision, Springer, 2021. (Best Paper Award) (Link)
